ExtractorSharp 
==
游戏资源编辑器

文件支持
---
   + NPK
   + SPK
   + IMG
        + Ver2
        + Ver4
        + Ver5
        + Ver6
        
依赖
---
   + 音频处理
       + Bass
   + 压缩
       + Zlib
       + ICSharpCode.SharpZipLib
   + 控件
       + WinForm
   + 插件支持
       + MEF
       
开发环境
---
   + Windows 10
   + .NET Framework 4.6.1
     
运行环境
---
   + Windows 7,8,10
   + .NET Framework 4.6 以上
**在Windows7环境中,你需要自行安装.NET Framework 4.6**
   
国际化支持(Language)
---
   ExtractorSharp的默认语言是中文(简体),并自带了一个英文的语言文件。
    请打开设置(Setting)-语言(Language)切换
